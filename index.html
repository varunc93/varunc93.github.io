Gesture Recognition System for American Sign Language

Matlab

The aim of this project was to decipher the meaning of sentences communicated in American Sign Language (ASL). The initial data for a set of words were collected using with MYO band (worn on forearm) giving raw Electromyography (EMG) & Orientation data used to identify the finger gestures. Feature extraction techniques like RMS, variance, etc were used, followed by transformation using PCA. The final output is fed to machine learning classification models like SVM, neural nets, and decision trees.
